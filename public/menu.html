📋 Game Instructions

🎯 Objective:

Get the highest score possible in 50 spins

🎰 Symbols and Scores:

  • 3 x 7️⃣ = 300 points (Jackpot - 1 in ~35 spins)
  • 3 x ⭐ = 270 points (Wild completes any combination)
  • 3 x 💎 = 150 points
  • 3 x 🔔 = 100 points
  • 3 x 🍇 = 80 points
  • 3 x 🍋 = 60 points
  • 3 x 🍒 = 40 points
  • 3 x 🎰 = 25 points + 3 free spins
  • 2 same + any = Variable points (2-20)
  • 3 x 💀 = -25 points (very rare)
  • 3 x ☠️ = -10 points (very rare)

🎁 Extra Spins:

  • 🎰🎰🎰 (3 slot machines) = +3 free spins
  • 💎💎 (2 diamonds) = +1 free spin
  • ⭐ (wild in center) = +1 free spin

📊 Probabilities (50 spins):

  • 7️⃣ appears ~35% per reel (more frequent!)
  • Negative symbols: ~3% total per reel (less punishing)
  • Wild (⭐) appears ~4% per reel
  • Expected ~20+ wins per 50 spins
  • Jackpot probability: ~75% in 50 spins
