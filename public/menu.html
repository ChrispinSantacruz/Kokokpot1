📋 Game Instructions

🎯 Objective:

Get the highest score possible in 30 spins

🎰 How it Works:

  • Every spin always shows 3 identical symbols!
  • You will always win points or lose points
  • No more "no result" spins!

🎰 Symbols and Scores:

  • 3 x 7️⃣ = 300 points (Jackpot)
  • 3 x ⭐ = 270 points (Wild)
  • 3 x 💎 = 150 points (Diamond)
  • 3 x 🔔 = 100 points (Bell)
  • 3 x 🍒 = 40 points (Cherry)
  • 3 x 💀 = -25 points (Negative)
  • 3 x ☠️ = -10 points (Negative)

🎁 Extra Spins:

  • 3 x 💎 (3 diamonds) = +1 free spin
  • 3 x ⭐ (3 wilds) = +2 free spins

📊 Game Rules:

  • Every spin guarantees a result!
  • Always 3 identical symbols appear
  • You either win points or lose points
  • Maximum 10 free spins per game
  • Game ends when you run out of spins

🎮 How to Play:

  • Click the SPIN button or pull the lever
  • 3 matching symbols will always appear
  • Watch out for negative symbols!
  • Use free spins wisely

Good luck!
